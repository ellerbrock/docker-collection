## Installation

`docker pull frapsoft/alpine-sass`

## Example

```
docker run -it \
  -v ${PWD}/sass:/sass \
  -v ${PWD}/css:/css \
ellerbrock/alpine-sass /sass/main.sass /css/main.css
```
